# Custom loading screen example templates

1920×1080 PNG templates referenced by the [Custom Assets](../../docs/wiki/features/custom-assets.md) wiki article.

Regenerate:

```bash
python3 scripts/generate-loading-screen-examples.py
```

On `make debug` / `make release`, these files are copied to `dist/debug/` or `dist/prod/` next to the built mod DLL.
